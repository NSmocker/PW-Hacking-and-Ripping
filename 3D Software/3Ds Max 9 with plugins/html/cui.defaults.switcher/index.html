Configure 3ds max's options to match the type of work you will be doing:

Artists and designers in different industries use 3ds max’s tools in different ways.

The Initial settings for tool options control the initial settings of the tools in 3ds max, to more closely match the type of work you are doing, while the UI schemes options control how the user interface of 3ds max will look.

Click on an item in the list on the left above to choose a group of initial tool settings most suited to your type of work.

Click on an item in the list on the right to choose a tool layout you prefer.

Click on any item to get more information about that choice – you have not committed to that choice until you click the Set button below.

Note: The 3ds max's tutorials assume that you are using the DefaultUI interface scheme and the Max initial settings (defaults).
